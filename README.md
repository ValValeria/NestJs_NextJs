# Chat app
Simple template of chat app. 

## Running the app
Firstly, you have to create a file, named `config.ts`, which must be located inside `server2/src`.
Then type in the terminal the following
```bash
# development
$ cd server2
$ npm install
# watch mode
$ npm run start:dev
```
In the separate terminal
```bash
# development
$ cd client
$ npm install
$ npm run build
# watch mode
$ npm run start
```
